What is CI/CD?

CI/CD is a way to automate software development tasks like:

Building code
Testing code
Deploying applications
CI = Continuous Integration

Whenever developers push code to GitHub:

Code is automatically checked
Tests run automatically
Errors are found early
CD = Continuous Delivery / Deployment

After testing:

Application is automatically deployed to a server/cloud

Think like this:

Developer writes code
        ↓
Pushes code to GitHub
        ↓
GitHub Actions runs automatically
        ↓
Build + Test + Deploy
